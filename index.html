🔴 必须新建
① 模型验证 → 固有动力学
先确认CFD数值方法可靠，才能相信自由衰减给出的固有频率。若模型不可信，后续所有结论均无效。网格无关性验证是整个研究的地基。
② 理论模型 ↔ 固有动力学
解析公式的 f_theory 必须与CFD自由衰减的 f_CFD 对比，误差在5%内则相互支撑；差异本身也可能是非线性效应的新发现，是论文最核心的创新支撑。
③ 固有频率 → 频响扫频中心
自由衰减得到的 f_n 决定了强迫振荡应在哪个频段加密扫描。现有0.5~2.5Hz范围必须以 f_n 为中心展开，才有理论依据，否则审稿人会质疑选频依据。
④ 频响特性 → 极限承压
共振频率处振幅最大，是最危险工况。极限承压测试应优先在共振频率下进行，建立"频率-幅值-破坏压力"三维关系，才能给出完整安全包络线。

1.1 网格无关性验证 — 增强版 ✓ 已完成基础 ⚠ 需增强

›
  • A · 已完成：三套网格对比 (3mm / 2mm / 1.2mm)
    现有工作已覆盖三个网格尺度的对比，是合格的起点。
  • B · 【必须】引入Richardson外推法计算GCI指数
    从三套网格的特征量（水封最大振幅、液面压力峰值）计算网格收敛指数GCI，证明细网格误差＜1%。这是SCI审稿人的基本要求。
    GCI = Fs × |ε| / (r^p - 1)
    Fs=安全系数(1.25)，r=网格细化比，p=收敛阶数，ε=相对误差
  • C · 【必须】补充多指标收敛验证（不能只看水面高度）
    同时报告以下三个物理量的收敛性：① 水封振幅(mm)，② 气液界面位置(相对误差)，③ 气侧压力波动幅值(Pa)。制作"三指标-三网格"收敛曲线图。
  • D · 【必须】湍流模型敏感性对比 (k-ε vs k-ω SST)
    在最优网格下对比两种湍流模型对自由液面捕捉的差异，选择最优模型并给出理由。
    ⚠ 审稿人必问：为何选该湍流模型？若无此对比极可能被要求大修。
  • E · 【建议】钟罩式：近壁区 y+ 值验证
    钟罩式内外筒间隙处流速较高，需验证近壁节点y+值满足所选湍流模型要求，并给出y+分布云图。
输出：GCI表 + 三指标收敛图 + 湍流模型对比表 支撑：确立后续所有仿真的网格方案与湍流模型
⚡ 阶段二
解析理论模型推导（核心创新点之一）
🔴
为何必须先做理论推导？ SCI论文不是仿真报告。CFD只能告诉你"是什么"，理论模型才能解释"为什么"。若没有解析公式，审稿人会认为研究只是参数扫描，缺乏学术深度，极难发表于Q1期刊。理论公式与CFD的比对是论文最核心的创新支撑，且推导工作与CFD补充可并行，不浪费时间。
2.1 U型弯固有频率解析模型推导 ∑ 理论推导 🔴 必须新增
📁 纸笔推导 + MATLAB/Python验证 ⏱ 5~7天 🔗 前置：1.1
›
  • A · 建立U型弯液柱振荡运动方程
    将U型弯水封视为等截面液柱，忽略表面张力，考虑重力和流体摩擦阻力，建立类谐振子的二阶常微分方程。
    ρ·L_eff·ẍ + c·ẋ + 2ρg·x = 0
    无阻尼固有频率：f_n = (1/2π)·√(2g/L_eff)
  • B · 确定等效液柱长度 L_eff 的几何计算方法
    推导L_eff与几何参数（管径D、水封深度H、弯管曲率半径R）的关系表达式，这是理论模型的核心参数。
    💡 参考：Swaffield & Galowin (1992) 液柱振荡模型，以及 Wong et al. 排水管道压力传播模型。
  • C · 推导等效阻尼系数 c 的表达式
    阻尼来源：① 管壁摩擦（Hagen-Poiseuille流动阻力），② 弯管局部阻力（引入局部阻力系数ξ）。合并为等效线性阻尼，计算阻尼比ζ_theory。
  • D · 与CFD自由衰减结果对比制表
    将理论计算的 f_n 与任务3.1 CFD得到的 f_CFD 制成对比表，计算相对误差。误差在5%以内则两者相互验证。
    🔑 这一对比表是论文中最有说服力的结果之一，必须出现在论文正文。
输出：U型弯解析公式 + 阻尼比理论预测值 + CFD对比表 创新：提供可用于工程设计的显式计算公式
2.2 钟罩式存水弯固有频率修正模型推导 ∑ 理论推导 🔴 必须新增
📁 纸笔推导 + CFD验证 ⏱ 5~7天 🔗 前置：2.1
›
  • A · 分析钟罩式截面变化特征
    钟罩式沿程截面积不均匀（内筒、间隙、底部各不同），不能直接套用U型弯公式，需引入截面积加权修正。
  • B · 推导面积加权等效液柱长度 L*_eff
    将钟罩式沿流程分段，每段截面积 A_i，长度 l_i，推导等效液柱长度。这是相比U型弯公式的核心修正项。
    L*_eff = A_ref × Σ(l_i / A_i)
    f*_n = (1/2π)·√(2g·A_ref / Σ(A_ref²·l_i/A_i))
  • C · 钟罩节流效应对阻尼的额外贡献
    钟罩式内外筒间隙处存在显著节流效应（面积突变→涡耗散），需在阻尼模型中额外引入节流阻力项，这是钟罩式阻尼比高于U型弯的物理根源。
    🔑 这一机理分析是两种存水弯对比研究的核心发现，必须用VOF流场的涡量图加以可视化验证。
输出：钟罩式修正公式 + U型 vs 钟罩式公式对比表 创新：首次推导含截面修正的钟罩式固有频率解析式

3.1 固有动力学辨识（自由衰减）— 深化版 ✓ 基础已有 ⚠ 需大幅增强
📁 U型弯 + 钟罩式 ⏱ 5~8天 🔗 前置：1.1、2.1、2.2
›
  • A · 已完成：单一幅值自由衰减（±50mm）
    已获得基础振荡频率，是良好起点。需在此基础上补充以下内容。
  • B · 【必须】多幅值自由衰减 — 非线性效应探测
    增加初始扰动幅值：±10mm、±30mm、±80mm（共4个幅值）。绘制"初始幅值 vs 固有频率"图，若频率随幅值变化，说明系统存在非线性刚度特征。
    ⚠ ±80mm扰动可能接近水封破坏临界，需谨慎设置并缩短时间步。
  • C · 【必须】从衰减曲线提取阻尼比 ζ
    用包络线法（相邻峰值比）计算对数衰减率δ，进而得到阻尼比ζ。重点给出U型弯 vs 钟罩式的ζ对比，建议用连续5个峰值取均值。
    δ = ln(A_n / A_{n+1})，ζ = δ / √(4π² + δ²)
  • D · 【必须】流场可视化 — 阻尼机制解释
    截取关键时刻（最大正位移/零位移/最大负位移）的流场快照，输出VOF相云图、速度矢量场、涡量云图。重点对比U型弯与钟罩式在节流处的涡耗散差异。
  • E · 【必须】制作两种存水弯固有动力学参数对比表
    包含：固有频率f_n（理论/CFD/实验三值）、阻尼比ζ（理论/CFD/实验三值）、等效液柱长度L_eff。所有结论必须有定量数字支撑。
    🔑 定量对比表是论文中说服审稿人的核心图表之一。
输出：阻尼比提取结果 + 非线性幅频图 + 流场可视化 + 参数对比表 支撑：确定任务3.2强迫振荡的频率扫描中心点
3.2 频率响应特性（强迫振荡）— 深化版 ✓ 基础已有 ⚠ 需大幅增强
📁 U型弯 + 钟罩式 ⏱ 8~12天（仿真量最大） 🔗 前置：3.1完成
›
  • A · 已完成：7个频率点单幅值强迫振荡
    已有0.5, 1.0, 1.5, 1.6, 1.7, 2.0, 2.5 Hz的仿真结果，共14个工况。需在此基础上系统扩充。
  • B · 【必须】确定激励压力幅值依据，进行多幅值扫描
    引用GB 50015规范确定低/中/高三个幅值（建议±10Pa、±30Pa、±80Pa），在三个幅值下各重复全频率扫描，判断系统线性性。
    💡 若高幅值下频响曲线出现不对称或峰值漂移，说明系统具有非线性特性（Duffing型），这是极有价值的发现。
  • C · 【必须】共振区精细扫描（步长0.05Hz）
    在共振频段以0.05Hz步长精细扫描，精确定位共振峰，测量半功率带宽计算品质因数Q值，与阻尼比ζ进行一致性验证（Q = 1/(2ζ)）。
  • D · 【必须】构建完整FRF曲线与伯德图（Bode Plot）
    提取三类响应量：① 水封振幅(mm)；② 气侧压力传递比(无量纲)；③ 响应与激励相位差(°)。振幅+相位上下排列即为伯德图。
    🔑 伯德图在建筑排水领域是首创性分析工具引入，是本论文方法论层面的创新点，需在论文中明确指出。
  • E · 【重要】共振工况下的流场时序分析
    在共振频率激励下，以1/8周期为间隔输出完整周期内8帧流场快照（VOF界面演变、速度场、涡量场），重点捕捉气泡夹带、界面撕裂等破坏前兆特征。
  • F · 【重要】真实排水压力谱输入验证
    收集或引用文献中建筑立管实测压力波动数据（宽谱信号），代替纯正弦激励，进行至少1个工况的宽谱响应仿真，验证简谐激励结论的适用范围。
    ⚠ 若无实测数据，可引用EN 12056或GB 50015中规定的设计压力波动范围构造宽谱信号。
输出：完整FRF曲线 + 伯德图 + 非线性幅频图 + 共振流场时序图 支撑：确定任务3.3极限承压测试的最危险工况
3.3 极限承压能力（压力阈值）— 深化版 ✓ 基础已有 ⚠ 需大幅增强
📁 U型弯 + 钟罩式 ⏱ 6~10天 🔗 前置：3.2完成
›
  • A · 已完成：线性递增压力下的水封破坏测试
    已获得初步破坏压力阈值，是良好起点，需补充以下关键内容。
  • B · 【必须】明确水封失效判据（定量化）
    必须采用至少一种定量判据：① VOF判据：气相体积分数α_gas超过0.5；② 几何判据：水封最小厚度降至初始深度10%；③ 压力穿透判据：气体压力传递比突然超过0.9。
    🔑 明确的失效判据是实验与数值结果可对比的前提，也是论文可重复性的保证。
  • C · 【必须】三种加载制度对比研究
    ① 准静态加载（0.1 Pa/s缓慢递增）→ 静态极限；② 动态线性加载（10 Pa/s 和 50 Pa/s）→ 速率效应；③ 冲击加载（阶跃压力）→ 模拟卫生器具冲水瞬态。记录每种工况的破坏压力P_f和破坏时间t_f。
  • D · 【重要】共振频率下的极限测试
    在共振频率下施加递增幅值正弦压力，记录临界振幅P_res，与非共振工况对比，量化共振对承压能力的削弱倍数。
    ⚠ 预期：共振频率下的破坏压力阈值将远低于准静态值，这一发现将直接转化为设计建议。
  • E · 【重要】疲劳效应研究 — 循环加载后的承压衰减
    先进行N次（N=10, 50, 100, 500次）强迫振荡循环，再进行极限承压测试。绘制"循环次数 vs 极限承压阈值"曲线，评估水封"疲劳破坏"特征。
    🔑 这是存水弯研究领域几乎未被触及的方向，有望成为论文最具创新价值的贡献之一。
  • F · 【建议】构建动态安全包络曲面
    将所有工况的（激励频率, 激励幅值, 破坏时间）整理为三维数据，拟合成"动态承压包络曲面"，提出工程安全系数建议值（以GB 50015规范最大设计压力为基准）。
输出：P-t曲线 + 三加载对比表 + 疲劳衰减曲线 + 安全包络图 价值：直接转化为设计规范中存水弯选型安全系数建议

📁 以U型弯为主，钟罩式验证 ⏱ 8~15天 🔗 前置：3.1、3.2完成

🔴 阶段四
物理实验验证（SCI发表的必要条件）
🚨
为何实验验证是必须项而非可选项？ 在 Building and Environment 等顶级期刊，纯数值研究必须提供实验验证，否则审稿人会以"数值方法未经验证"为由拒稿。实验无需覆盖所有工况，但至少需验证自由衰减和强迫振荡两类基本工况，以证明CFD模型的预测精度。
4.1 实验台设计与搭建 🔴 必须新建 🔬 实验
⏱ 10~20天（含采购） 🔗 前置：3.1完成（已知f_n才能设计激励装置）
›
  • A · 实验台核心组件设计
    ① 透明亚克力U型弯和钟罩式各一个；② 气压激励装置（精密气泵+比例阀+信号发生器，0~10Hz，0~200Pa）；③ 压力传感器×2（精度±0.1Pa）和液位传感器×2（量程0~200mm，精度±0.5mm）；④ 数据采集系统（采样率≥100Hz）。
    💡 若实验室条件有限，可仅搭建最简版：透明存水弯 + 手动注水实现初始扰动（自由衰减实验），已足以验证固有频率的CFD预测。
  • B · 实验工况设计原则
    优先验证3类关键工况：① 自由衰减（验证f_n和ζ）；② 共振频率处的强迫振荡（验证最危险工况）；③ 准静态极限承压（验证P_max）。选择CFD结果中"最有特征性"的工况而非最多工况。
输出：实验台实物照片 + 测量系统示意图（论文Fig.1标配）
4.2 实验数据采集与CFD比对验证 🔴 必须新增 🔬 实验
⏱ 5~10天 🔗 前置：4.1完成
›
  • A · 自由衰减实验 — 固有频率与阻尼比验证
    对两种存水弯各进行3次重复实验，用FFT分析液位时程提取实验固有频率f_exp，计算实验阻尼比ζ_exp，与CFD和理论值列表对比，报告平均误差和标准差。
  • B · 强迫振荡实验 — FRF曲线验证
    在0.5~2.5Hz范围内测量实验FRF曲线，与CFD结果叠加绘制对比图，检验共振峰位置（误差＜5%）、振幅（误差＜20%）及相位特性。曲线对比图要同时画出误差棒（error bar）。
